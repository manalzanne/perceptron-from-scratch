# Perceptron from Scratch

Implementation of the Perceptron algorithm from scratch using Python and NumPy.

## Technologies Used
- Python
- NumPy
- Matplotlib

## Project Steps
1. Creation of logical gate datasets (AND, OR, XOR)
2. Implementation of the activation function
3. Weight initialization
4. Perceptron training loop
5. Decision boundary visualization
6. Demonstration of the XOR limitation

## Results
The perceptron successfully learns linearly separable problems such as AND and OR, but fails to solve XOR, illustrating a fundamental limitation of single-layer neural networks.
